# Warehouse

This repository contains the **Warehouse Operations Application** — a full-stack WMS for fragrance and cosmetics manufacturing.

## Getting Started

All application code, setup instructions, demo accounts, and RBAC documentation live in:

**[warehouse-app/README.md](./warehouse-app/README.md)**

### Quick run

```bash
# Terminal 1 — API (port 3001)
cd warehouse-app/backend && npm install && npm run dev

# Terminal 2 — UI (port 3000)
cd warehouse-app/frontend && npm install && npm run dev
```

Open http://localhost:3000 and log in with any demo account (password: `password123`). See the full README for roles and walkthroughs.
